# input()函数
判断是否为闰年示例：
```
year = int(input('请输入年份: ')) #传入的类型是字符串，需要强制类型转换

# 如果代码太长写成一行不便于阅读 可以使用\或()折行
if (year % 4 == 0 and year % 100 != 0 or year % 400 == 0):
    print("%d是闰年" % year)
else:
    print("%d不是闰年" % year)

a = 10
print(isinstance(a,int))#判断a变量的类型是否为int
```
